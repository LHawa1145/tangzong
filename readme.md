这是一个学习前端开发的测试网站项目。

## 项目地址
- GitHub: https://github.com/LHawa1145/tangzong
- 在线访问: https://lhawa1145.github.io/tangzong/

## 项目结构
```

tangzong/
├──index.html      # 主页面
├──style.css       # 样式表
├──kefu1.png       # 客服图片1
├──kefu2.png       # 客服图片2
└──README.md       # 项目说明（本文件）
```

## 技术栈
- HTML5
- CSS3
- GitHub Pages

## 功能
- 响应式导航栏
- 客服信息展示
- 外部链接跳转

## 本地运行
1. 克隆仓库：`git clone https://github.com/LHawa1145/tangzong.git`
2. 打开index.html文件即可
